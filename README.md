本项目大量使用deepseek生成的代码
## 主要功能特点：
1. **电表数据获取** - 支持重试3次机制
2. **数据存储** - SQLite数据库存储历史数据
3. **邮件提醒** - 带时间延迟和重试机制的邮件发送
4. **网页展示** - 自动生成HTML监控页面
5. **低电量警报** - 阈值触发邮件通知
6. **日志记录** - 完整的运行日志
7. **优雅退出** - Ctrl+C安全退出

### 🔧 编译命令：
```bash
gcc -o 电表监控.exe 电表监控.c -lwininet -lsqlite3 -lws2_32
```

### 📧 邮件发送优化：
- **收件人延迟**：每个收件人之间3秒间隔
- **重试机制**：失败后递增延迟重试（5s, 10s, 15s）
- **稳定连接**：使用 System.Net.Mail.SmtpClient
- **资源管理**：正确释放所有资源

### ⚡ 使用建议：
1. 确保 `config.txt` 配置文件正确
2. 检查邮箱SMTP服务已开启并获取授权码
3. 程序会持续监控并按设定间隔运行
4. 按 Ctrl+C 可以安全退出程序
